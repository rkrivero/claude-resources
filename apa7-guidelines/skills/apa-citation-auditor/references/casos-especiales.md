# Casos especiales de referencia — APA 7, Cap. 10 (ampliacón)

## Árbol de decisión — identificación del tipo de fuente

```
¿Es un artículo revisado por pares en una revista académica?
  → Sí → artículo de revista (con DOI si existe)
  → No ↓

¿Es un capítulo dentro de un libro con editor distinto al autor del capítulo?
  → Sí → capítulo en libro editado
  → No ↓

¿Es un documento de política, técnico o estadístico emitido por organización?
  → Sí → informe institucional / reporte
  → No ↓

¿Es software, dataset o shapefile?
  → Sí → software / conjunto de datos
  → No ↓

¿Es una tesis o disertación?
  → Sí → tesis
  → No ↓

¿Es un preprint (sin revisión de pares todavía)?
  → Sí → artículo preprint
  → No → página web / entrada de blog / recurso online
```

## Actas de congreso / conferencia

### Trabajo publicado en actas impresas o con DOI

```
Autor, A. A. (Año). Título de la ponencia. En E. Editor (Ed.),
    Título de las actas (pp. xx–xx). Editorial.
    https://doi.org/xxxxx
```

```
Montoya, P. (2022). Segregación residencial y accesibilidad en ciudades
    patrimoniales. En M. García (Ed.), Actas del XII Congreso de Morfología
    Urbana (pp. 134–149). ISUF-H. https://doi.org/10.xxxxx/xxxxx
```

### Ponencia sin publicar / presentación en congreso

```
Autor, A. A. (Año, Mes). Título de la presentación [Tipo de presentación].
    Nombre del congreso, Ciudad, País.
```

```
Gómez, R. (2023, noviembre). Análisis de accesibilidad espacial en zonas
    urbanas mediante SIG [Presentación de póster]. XIV Congreso
    Latinoamericano de Urbanismo, Santiago, Chile.
```

---

## Preprints

```
Autor, A. A. (Año). Título del artículo. Nombre del servidor de preprints.
    https://doi.org/xxxxx
```

```
Rodríguez, M., & Salas, C. (2024). Spatial clustering and inequality in
    Latin American cities: A LISA approach. SocArXiv.
    https://doi.org/10.31235/osf.io/xxxxx
```

Nota en texto: indicar que es preprint si es relevante:
`(Rodríguez & Salas, 2024, preprint)`

---

## Artículos en prensa / advance online publication

```
Autor, A. A. (en prensa). Título. Nombre de la Revista.
    https://doi.org/xxxxx
```

Cita en texto: `(Flores, en prensa)`

## Informes de organismos internacionales

```
ONU-Hábitat. (2022). Informe mundial sobre las ciudades 2022.
    Programa de las Naciones Unidas para los Asentamientos Humanos.
    https://unhabitat.org/wcr/2022
```

### UNESCO (Patrimonio)

```
UNESCO. (2011). Recomendación sobre el paisaje urbano histórico.
    Organización de las Naciones Unidas para la Educación, la Ciencia
    y la Cultura. https://whc.unesco.org/uploads/activities/documents/
    activity-638-98.pdf
```

### Banco Mundial / BID

```
Banco Mundial. (2021). Ciudades del futuro: Planificación urbana en
    América Latina. Grupo del Banco Mundial. https://doi.org/10.xxxxx
```

### CEPAL

```
Comisión Económica para América Latina y el Caribe. (2020). Título
    del informe (No. LC/XXX). CEPAL. https://repositorio.cepal.org/url
```

## Legislación y normas legales peruanas

APA 7 adapta el formato del Bluebook. Para Perú:

### Ley

```
Ley N.° XXXXX. Nombre oficial de la ley. Congreso de la República
    del Perú. Fecha de promulgación. Diario oficial El Peruano.
    https://www.gob.pe/normas-legales/...
```

### Decreto Supremo

```
Decreto Supremo N.° XXX-XXXX-PCM. Nombre o asunto. Presidencia del
    Consejo de Ministros. Fecha. El Peruano. https://url
```

Cita en texto: `(Ley N.° 29090, 2007)` o `(D.S. N.° 022-2016-MTC, 2016)`

## Capas GIS / shapefiles / datos geoespaciales institucionales

```
Institución Productora. (Año). Nombre de la capa o dataset [Capa SIG /
    Conjunto de datos geoespaciales]. Plataforma o repositorio.
    https://url
```

Ejemplos de capas GIS institucionales:

```
[Institución cartográfica nacional]. ([Año]). [Nombre de la capa — descripción
    temática y alcance geográfico] [Conjunto de datos geoespaciales].
    [Nombre del geoportal o repositorio]. https://url-del-repositorio

[Organismo estadístico]. ([Año]). [Shapefile / capa] — [ámbito geográfico]
    [Conjunto de datos geoespaciales]. [Institución].
    https://url-del-repositorio

[Agencia de infraestructura]. ([Año]). [Red vial / transporte / servicios]
    [Conjunto de datos geoespaciales]. [Geoportal institucional].
    https://url-del-geoportal
```

## Retracciones

Si se cita un artículo retractado (solo cuando es esencial):

```
Autor, A. A. (Año). Título [Artículo retractado]. Nombre de la Revista,
    vol(núm), pp–pp. https://doi.org/xxxxx (Artículo original publicado
    en Año)
```

Cita en texto: `(Autor, año, retractado)`

---

## Traducciones

```
Apellido, N. (Año). Título en español (T. Traductor, Trad.; N.ª ed.).
    Editorial. (Obra original publicada en año)
```

```
Lefebvre, H. (2013). La producción del espacio (E. Martínez Gutiérrez,
    Trad.). Capitán Swing. (Obra original publicada en 1974)
```

Cita en texto: `(Lefebvre, 1974/2013)`

---

## Entradas de diccionario o enciclopedia (online, con autor)

```
Autor, A. A. (Año). Entrada. En Nombre de la enciclopedia. Editorial/sitio.
    https://url
```

Sin autor → título de la entrada sube a posición de autor.
