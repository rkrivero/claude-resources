# Criterios de legibilidad lingüística — Cassany (1993)

Fuente: Cassany, D. (1993). *La cocina de la escritura* (Cap. 1, "La legibilidad" y "El estilo llano"). Anagrama, Barcelona. Basado en: Flesch (1949), Henry (1987), Richaudeau (1984), Miller (1969).

---

## Tabla de legibilidad alta vs. baja

| Legibilidad ALTA | Legibilidad BAJA |
|---|---|
| Palabras cortas y básicas | Palabras largas y complejas |
| Frases cortas (≤ 20 palabras) | Frases largas (> 30 palabras) |
| Lenguaje concreto (personas, objetos, acciones) | Lenguaje abstracto |
| Estructuras que favorecen la anticipación del lector | Subordinadas e incisos demasiado largos |
| Redundancia controlada (repetición útil) | Enumeraciones excesivas sin estructura |
| Marcadores textuales que guían la lectura | Palabras importantes enterradas al final de la oración |
| Verbo en posición lógica (cerca del sujeto) | Monotonía estructural |
| Variación tipográfica útil: cifras, negrita, cursiva | — |

## Sistema de medición — puntos y criterios

| Punto medido | Sistema de medida |
|---|---|
| **Extensión del vocablo** | Número de sílabas por 100 palabras |
| | Número de letras, vocales o consonantes |
| **Vocabulario básico** | Número de palabras que no pertenecen a un vocabulario de referencia |
| | Número de afijos cultos (re–, in–, –ismo, –logía, –ción...) que indican palabras complejas |
| | Grado de variación léxica (más variedad = mayor probabilidad de palabras difíciles) |
| **Extensión de la oración** | Número de palabras por frase |
| | Cantidad de puntuación fuerte (punto, dos puntos, punto y coma, paréntesis) |
| | Número de preposiciones por oración (más preposiciones = mayor complejidad) |
| **Interés y concreción** | Número de mayúsculas que no empiezan oración (nombres propios) |
| | Palabras personales (pronombres, sustantivos de género natural: autor, investigador, ciudad) |
| | Puntuación activa (interrogaciones, exclamaciones, guiones) |
| | Oraciones en estilo directo, diálogos, citas literales |

## Parámetros de referencia (Miller, 1969; Richaudeau, 1984)

| Tipo de texto | Nivel de legibilidad |
|---|---|
| Cómics, libros de lectura básica, best-sellers | Muy alta |
| Periodismo generalista, manuales de uso | Alta |
| Artículos de divulgación científica | Media-alta |
| Tesis doctorales, artículos científicos | Media |
| Documentación legal, normativas | Media-baja |
| Literatura de élite, artículos muy especializados | Baja |

## Criterios del escrito llano eficaz (Cassany, 1993)

Un texto académico en estilo llano reúne estas 4 condiciones:

**1. Lenguaje apropiado al lector y al documento**
Calibrar el nivel técnico según las necesidades y el conocimiento previo del lector
específico, y según el género del documento. Las ponencias para especialistas usan
terminología técnica que los resúmenes ejecutivos no usan.

**2. Diseño racional**
Los datos relevantes en posiciones prominentes: inicio del párrafo, inicio de la
oración. La "letra pequeña" informativa no va enterrada en subordinadas o paréntesis.

**3. Comprensible en primera lectura**
Si el lector debe detenerse para releer o reconstruir la sintaxis, el texto falla.
Test: ¿puede seguirse el hilo sin volver atrás?

**4. Cumple requisitos del género**
La simplificación no puede comprometer la precisión técnica ni el hedging académico
necesario en el género científico.

## Los dos obstáculos insalvables de comprensión

Cassany, basándose en la investigación del estilo llano, identifica que los obstáculos
que el lector **no puede superar solo** son:

**Obstáculo 1: Estructura sintáctica excesivamente compleja**
Subordinadas anidadas, períodos largos, incisos extensos. El lector pierde el hilo
sintáctico y necesita ayuda externa para reconstruirlo.

**Obstáculo 2: Ausencia de contexto compartido**
Texto que asume conocimiento previo que el lector no tiene, sin proporcionar el andamio
conceptual necesario. El lector queda sin punto de apoyo.

**Por qué el léxico técnico no es insalvable:**
Un término desconocido puede buscarse en un glosario o referencia. Una sintaxis rota
no puede reconstruirse sin el autor. → Priorizar simplificación sintáctica.

## Aplicación a secciones de tesis

| Sección | Audiencia típica | Legibilidad objetivo | Observaciones |
|---|---|---|---|
| Abstract | Investigadores del campo y comité evaluador | Media-alta | Densidad conceptual necesaria; frases claras imprescindibles |
| Introducción | Investigadores y lector académico general | Media-alta | Más contexto concreto que en el desarrollo |
| Marco teórico | Especialistas del área | Media | Tecnicismos necesarios; frases no deben superar 30 palabras |
| Metodología | Especialistas que replican el estudio | Media | Claridad procedimental prioritaria |
| Resultados | Especialistas que evalúan | Media | Datos concretos anclan las afirmaciones abstractas |
| Discusión | Investigadores del campo | Media | Argumentación; hedging; conexión con literatura |
| Conclusiones | Audiencia amplia del campo | Media-alta | Síntesis en lenguaje accesible |
