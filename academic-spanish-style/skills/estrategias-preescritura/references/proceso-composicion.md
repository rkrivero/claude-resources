# El proceso de composición — subprocesos y tipos de escritura académica

Fuente: Cassany, D. (1993). *La cocina de la escritura* (Caps. 1–2). Anagrama, Barcelona. Basado en: Flower & Hayes (1981), Murray (1978), Camps (1990).

---

## Los subprocesos de escritura

La escritura no es un proceso lineal (planificar → redactar → corregir). Es un proceso
recursivo en el que los subprocesos se alternan e interrumpen entre sí.

| Subproceso | Descripción | Técnicas asociadas |
|---|---|---|
| **Buscar ideas** | Generar contenido sin censura | Torbellino de ideas, Estrella de preguntas, Escritura libre |
| **Organizar ideas** | Estructurar y jerarquizar el contenido | Ideogramas, Mapas mentales, Esquemas |
| **Formular objetivos** | Definir qué se quiere lograr con el texto | Análisis de la situación comunicativa; revisar objetivos de la tesis |
| **Redactar** | Convertir el esquema en prosa | Borradores rápidos; señales para el lector |
| **Revisar** | Mejorar el texto a nivel de contenido y forma | Revisión multi-pase (ver `pases-revision.md`) |

---

## Diferencias experto vs. novato en el proceso global

| Dimensión | Escritor novato (aprendiz) | Escritor experto |
|---|---|---|
| Pre-escritura | Mínima o inexistente; empieza a redactar de inmediato | Extensa; planifica estructura y argumento antes de escribir |
| Borrador | Escribe para "rellenar" el espacio requerido | Escribe para desarrollar y clarificar ideas |
| Revisión | Superficial; corrección de errores de forma | Profunda; revisa contenido primero, forma después; múltiples pases |
| Conciencia del lector | Baja; el texto es para el autor | Alta; considera constantemente qué sabe y necesita el lector |
| Flexibilidad | Lineal; escritura = transcripción del pensamiento | Recursiva; vuelve a fases anteriores cuando el texto lo requiere |
| Bloqueo | Intentar escribir la versión perfecta de inmediato | Separar la generación de la evaluación |

## Las 3 fases del proceso de composición

```
PRE-ESCRIBIR             ESCRIBIR              RE-ESCRIBIR
(preedición)             (borrador)            (revisión)
     │                       │                      │
Torbellino de ideas    Primer borrador rápido  Pase 1: contenido
Estrella de preguntas  Sin autocorrección      Pase 2: forma
Escritura libre        Basado en el esquema    (ver revision-por-capas)
Ideograma / Esquema    ↓ rápido y continuo
     │
     ↓
Esquema jerárquico
```

**Principio de separación de fases:** El borrador debe escribirse rápido, sin
preocuparse por la perfección. La perfección se construye en la revisión. Separar
la fase de generación de la fase de evaluación reduce el bloqueo y mejora la calidad
final del texto.

## Tipos de escritura académica en la tesis (Cassany, 1993)

Adaptado de la tipología de Cassany para el contexto de tesis e investigación cuantitativa:

| Tipo | Propósito comunicativo | Secciones en tesis |
|---|---|---|
| **Expositivo** | Explorar y presentar información objetiva | Marco teórico, revisión de literatura, metodología, resultados |
| **Persuasivo** | Argumentar, defender una postura, justificar decisiones | Discusión, conclusiones, justificación metodológica, limitaciones |
| **Funcional** | Comunicar de forma estandarizada y económica | Abstract, resumen ejecutivo, tablas, figuras, notas al pie |

**Nota práctica para la tesis:** La mayoría de las secciones combinan lo expositivo y lo
persuasivo. El marco teórico no solo describe la literatura: *construye el argumento*
para que la pregunta de investigación tenga sentido. La metodología no solo describe
las técnicas: *justifica* por qué son las más adecuadas para los datos y la pregunta.

## La escritura como herramienta epistémica

**Principio de Cassany (1993):** Escribir no es solo comunicar lo que ya se sabe. Es
también una herramienta para *generar conocimiento*. Al escribir, se descubren conexiones
entre ideas, se identifican inconsistencias lógicas, se clarifican conceptos vagos.

Cassany: "Escribir es mucho más que un medio de comunicación: es un instrumento
epistemológico de aprendizaje. Escribiendo se aprende y podemos usar la escritura
para comprender mejor cualquier tema."

**Aplicaciones prácticas para la tesis:**
- Cuando un capítulo "no sale" después de varios intentos → probable que el argumento
  no esté aún suficientemente claro en el pensamiento del autor → volver a preedición
- El bloqueo de escritura no es un problema de estilo; es una señal de problema conceptual
- Escribir el borrador de la discusión antes de terminar de analizar los datos puede
  revelar qué análisis adicionales son necesarios

**Pregunta diagnóstica:**
¿Puedes explicar oralmente en 2 oraciones lo que quieres decir en este párrafo?

- Si SÍ → el problema es estilístico. Usar `revision-por-capas` o `legibilidad-estilo-llano`.
- Si NO → el problema es conceptual. Usar `estrategias-preescritura` T1 o T2.

## Equipo mínimo del escritor académico (adaptado de Cassany, 1993)

| Función | Herramienta |
|---|---|
| Gestión de referencias | Zotero, Mendeley (con plugin de Word/Overleaf) |
| Borrador y notas rápidas | Libreta física, Apple Notes, Notion |
| Escritura del documento | LaTeX (Overleaf), Word, Google Docs |
| Mapas mentales | papel + lápiz, draw.io, Miro, Obsidian Canvas |
| Verificación ortográfica | Corrector integrado + revisión manual final |
| Control de versiones | Git (LaTeX) o historial de versiones (Word/Google Docs) |

**Nota de Cassany:** No hay equipo correcto universal. El equipo del escritor es tan
personal como su proceso. Lo importante es que no entorpezca el pensamiento.
