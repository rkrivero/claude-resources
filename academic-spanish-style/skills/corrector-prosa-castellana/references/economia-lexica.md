# Economía léxica — palabras justas, no más

## Nominalizaciones — tabla de verbalizaciones

Transformar el "estilo nominal" (sustantivos abstractos apilados) en
"estilo verbal" (verbos que expresan acción):

| Estilo nominal | Estilo verbal |
|----------------|--------------|
| la realización de un análisis | analizar |
| la identificación de los factores | identificar los factores |
| la obtención de los datos | obtener los datos |
| la aplicación de la metodología | aplicar la metodología |
| la evaluación de los resultados | evaluar los resultados |
| el establecimiento de relaciones | establecer relaciones |
| la verificación de la hipótesis | verificar la hipótesis |
| la elaboración de modelos | elaborar modelos |
| la consideración de variables | considerar variables |
| la determinación del impacto | determinar el impacto |
| la utilización de herramientas SIG | usar herramientas SIG |
| la concretización del proceso | concretar el proceso |
| la implementación de la política | implementar la política → aplicar |
| la posibilidad de acceder | poder acceder |
| la capacidad de explicar | poder explicar / explicar |

**Señal de alerta:** tres o más sustantivos abstractos terminados en
-ción, -miento, -ización, -dad seguidos → volver al verbo.

## Verbos con sufijo -izar innecesarios

En general: si existe un verbo clásico de la misma raíz, usar ese.

| Neologismo -izar | Alternativa clásica |
|------------------|---------------------|
| concretizar | concretar |
| culpabilizar | culpar |
| recepcionar | recibir |
| influenciar | influir |
| posicionar (opinión) | tomar posición / posicionarse |
| visibilizar (cuando = mostrar) | mostrar / evidenciar / revelar |
| problematizar (cuando = cuestionar) | cuestionar / poner en duda |
| territorializar | circunscribir al territorio / aplicar al territorio |
| focalizar | centrar / enfocar |
| operacionalizar | operacionalizar (este SÍ es técnico y aceptado en metodología) |
| conceptualizar | conceptualizar (aceptado; no tiene alternativa exacta) |

**Nota:** algunos -izar son términos técnicos consagrados en ciencias sociales
(operacionalizar, territorializar en geografía crítica). Usar el criterio: ¿existe
una palabra más corta y exacta? Si sí → usarla.

## Palabras largas vs. palabras sencillas

Preferir la palabra corriente frente a la culta o formal cuando el significado es el mismo:

| Pedante / largo | Sencillo |
|-----------------|---------|
| aproximativo | aproximado |
| parágrafo | párrafo |
| finalizar | terminar / concluir |
| realizar (= hacer) | hacer / ejecutar |
| efectuar | hacer / realizar |
| vehicular (= transmitir) | transmitir / expresar |
| corolario (= conclusión directa) | consecuencia / conclusión |
| ulterior (= posterior) | posterior / siguiente |
| asimismo (cuando se repite mucho) | también / además |
| dicho (anáfora: "dicho autor") | este / ese autor |

## Muletillas y locuciones vacías — tabla de sustituciones

### Locuciones que se pueden eliminar directamente

| Muletilla | Acción |
|-----------|--------|
| de alguna manera | eliminar; si es necesario, especificar el modo |
| en cierto sentido | eliminar o especificar |
| el tema de / la cuestión de | eliminar y escribir directamente el sustantivo |
| el proceso de | eliminar cuando no agrega nada: "el proceso de análisis" → "el análisis" |
| es importante señalar que | eliminar; la oración siguiente lleva el peso |
| cabe destacar que | eliminar; reformular como afirmación directa |
| hay que tener en cuenta que | es necesario considerar que / [eliminar y decirlo] |
| a efectos de | para |
| en aras de | para / con el fin de |

### Locuciones que deben sustituirse

| Incorrecto / vacío | Correcto / preciso |
|--------------------|-------------------|
| en base a | basándose en / con base en / a partir de / según |
| a nivel de | en / entre / para / en el ámbito de |
| de cara a (finalidad) | para / con vistas a / con miras a |
| al objeto de | para / con el objeto de |
| bajo el punto de vista | desde el punto de vista |
| en lo que se refiere a | respecto de / en cuanto a / sobre |
| a nivel de los profesores | entre los profesores |
| a nivel estructural | en el plano estructural / estructuralmente |

| Calco del inglés | Español correcto |
|------------------|-----------------|

## Densidad verbal — equilibrio morfológico

Mantener un índice donde los verbos conjugados en formas personales igualen o
superen a los sustantivos abstractos de la frase. La acción se expresa con
dinamismo verbal, no con acumulación de nombres.

**Verbos predicativos vs. copulativos/débiles:** preferir verbos con significado
pleno frente a "ser/estar/haber + atributo" cuando exista una forma predicativa.

| Copulativo / débil | Predicativo específico |
|--------------------|------------------------|
| es indicativo de | indica / revela |
| está en relación con | se relaciona con / depende de |
| hay un aumento de | aumenta / se incrementa |
| es una muestra de | muestra / ejemplifica |
| tiene relación con | se vincula a |

### Perífrasis verbal de relleno

Eliminar el rodeo "auxiliar + infinitivo/gerundio" cuando una sola forma
conjugada basta:

| Perífrasis de relleno | Forma simple |
|-----------------------|-------------|
| viene a demostrar que | demuestra que |
| pasa a constituir | constituye |
| se procede a estimar | se estima |
| viene dado por | resulta de / es |
| va a depender de | depende de |

## Ratio léxica — palabras plenas vs. funcionales

Maximizar la proporción de palabras con carga semántica plena (sustantivos
específicos, verbos predicativos, adjetivos relacionales) frente a las
funcionales (artículos, preposiciones, conjunciones). El exceso de palabras
funcionales diluye la densidad del argumento.

```
Baja densidad: "El estudio que se ha hecho sobre el tema de la influencia de la
                accesibilidad en el precio del suelo de la ciudad."
                (muchas funcionales: que, se, ha, sobre, de, la, de, la, en, el…)
Alta densidad: "El estudio sobre la influencia de la accesibilidad en el precio
                del suelo urbano."
```

Señal de alerta: cadenas de "de + artículo + de + artículo" → reorganizar con
adjetivos relacionales (`del suelo de la ciudad` → `del suelo urbano`).

## Elipsis verbal condicionada

Elidir el verbo en cláusulas coordinadas **solo si** tiempo, modo, aspecto y
persona coinciden exactamente con el verbo antecedente. Si varían, conjugar:

```
Elipsis válida (mismo tiempo/persona):
"El OLS arrojó R² = .31; el SAR, R² = .52." (se elide "arrojó")

Elipsis inválida (cambia el tiempo):
"El modelo se estimó en 2023 y los resultados ahora disponibles."
Correcto: "El modelo se estimó en 2023 y los resultados están ahora disponibles."
```

## Control de adverbios en -mente

**Frecuencia máxima:** un adverbio en -mente por página. Su acumulación
fosiliza el texto y produce cacofonía.

**Sustituir por locuciones o reformular:**
| Adverbio -mente | Alternativa |
|-----------------|-------------|
| significativamente | de forma significativa / en grado significativo |
| considerablemente | en gran medida / de modo considerable |
| evidentemente | resulta claro que [reformular] |
| actualmente | en la actualidad / hoy |
| principalmente | sobre todo / ante todo |

**Dos adverbios -mente seguidos:** solo el último conserva la terminación:
"pura y simplemente" (no "puramente y simplemente").

**No abrir párrafos** sistemáticamente con "Posiblemente,", "Evidentemente,",
"Actualmente,".

## Monotonía léxica y eufonía

### Repetición de vocablos `[MONOTONIA]`
Prohibida la repetición de palabras con la misma raíz u homófonos en un radio
inferior a **tres líneas**. Resolver mediante elisión, pronominalización o
hiperónimo específico.

```
Repetido: "El modelo modela la relación. El modelado del precio depende del
           modelo espacial."
Resuelto: "El modelo capta la relación. La estimación del precio depende de
           su especificación espacial."
```

### Cacofonía y rima en prosa `[CACOFONIA]`
Interrumpir secuencias de palabras contiguas que terminen en las mismas sílabas
o fonemas. El texto científico debe ser rítmicamente neutro.

```
Rima involuntaria: "La estimación de la dispersión de la población."
Corregido:         "La estimación de cómo se dispersa la población."

Agudas seguidas:   "Evaluó la función según la cuestión central."
Corregido:         "Evaluó la función a partir de la pregunta central."
```

No acumular terminaciones verbales idénticas (`-ción`, `-ando`, `-mente`) ni
palabras agudas sucesivas en la misma línea.

## Diccionario de reemplazo — ciencia de datos espaciales

Reemplazos técnicos obligatorios para la tesis (perífrasis/comodín → término
monorreferencial). El término técnico es más corto y más preciso a la vez:

| Perífrasis / comodín | Reemplazo técnico |
|----------------------|-------------------|
| hacer la estimación de los parámetros | estimar los parámetros |
| el modelo da como resultado | el modelo reporta / genera / computa |
| tiene una influencia en | condiciona / determina / afecta |
| una característica propia del espacio | heterogeneidad espacial |
| el problema de la falta de datos | omisión de variables / sesgo muestral |
| llevar a cabo el proceso de web scraping | extraer datos mediante web scraping |
| factores topográficos | pendiente / altitud / configuración geomorfológica |
| a nivel de distrito | a escala distrital |
| la cuestión de la dependencia espacial | la dependencia espacial |
| el efecto que produce la accesibilidad | el efecto de la accesibilidad |
| zonas que están alejadas del centro | la periferia / las zonas periféricas |

**Principio:** en redacción técnica, el término especializado correcto suele ser
*más conciso* que el rodeo coloquial. La precisión y la economía coinciden.

## Técnicas para encontrar el verbo preciso

1. Pregunta: ¿qué hace exactamente el sujeto aquí? → buscar ese verbo
2. Si la respuesta contiene un sustantivo (-ción, -miento), nominalización presente → verbalizar
3. Si el verbo comodín ("hacer", "tener", "dar") se puede sustituir por la raíz del sustantivo siguiente → hacerlo
4. Para arquitectura/urbanismo: verbos técnicos preferibles:
   - zonificar, catastrar, delimitar, proyectar, trazar, articular, densificar,
     patrimonialiizar, revitalizar, consolidar, fragmentar, segregar
