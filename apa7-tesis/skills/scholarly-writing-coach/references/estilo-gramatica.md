# Estilo y gramática académica — APA 7, Cap. 4

## Tiempos verbales (Tabla 4.1 APA 7)

| Sección | Tiempo | Ejemplo correcto |
|---------|--------|-----------------|
| Literatura / trabajo de otros | Pasado | "Rosen (1974) propuso el modelo hedónico" |
| Literatura / trabajo de otros | Pret. perfecto | "Los investigadores han encontrado que…" |
| Descripción del método propio | Pasado | "Se recolectaron 487 registros de oferta" |
| Resultados | Pasado | "Los coeficientes fueron estadísticamente significativos" |
| Discusión — implicaciones | Presente | "Los resultados sugieren que la proximidad al CHC…" |
| Conclusiones / limitaciones | Presente | "La principal limitación del estudio es…" |
| Referencia a tablas/figuras del manuscrito | Presente | "La Tabla 3 muestra los coeficientes del modelo final" |
| Leyes universales / definiciones vigentes | Presente | "La función hedónica descompone el precio en atributos" |

## Voz activa vs. pasiva

Preferir activa cuando el investigador es el agente:
- Activa: "Filtramos los registros duplicados" ✓
- Pasiva: "Los registros duplicados fueron filtrados" — aceptable si el
  foco es el objeto

Pasiva aceptable:
- "Los shapefiles fueron reprojectados a EPSG:32719"
  (el foco es el resultado, no quién lo hizo)

## Pronombres de primera persona

- Solo/a: "I" → en español "yo" o forma verbal ("analicé", "identifiqué")
- Con coautores: "we" → "nosotros" o forma verbal plural
- NO: "el autor" / "la investigadora" cuando te refieres a ti mismo

## Noun strings — cómo desarmarlo

Regla: si hay ≥3 sustantivos modificando a un sustantivo final → deshacer.

```
Mal:  "análisis de accesibilidad urbana peatonal multiescala longitudinal"
Bien: "análisis longitudinal y multiescala de accesibilidad peatonal en
       entornos urbanos"
```

## Comparaciones lógicas

Los dos términos comparados deben ser del mismo tipo:

```
Mal:  "Los valores del CHC son mayores que las zonas periféricas"
Bien: "Los valores del CHC son mayores que los de las zonas periféricas"
Bien: "Los valores del CHC superan a los de las zonas periféricas"
```

## Antropomorfismo

| Incorrecto | Correcto |
|-----------|---------|
| El modelo concluye que… | Los resultados del modelo indican que… |
| El análisis sugiere que… | Con base en el análisis, se sugiere que… |
| La teoría afirma que… | La teoría postula / propone que… |
| Los datos muestran intención | Los datos reflejan / revelan |
| El estudio extrapoló la tasa | Extrapolamos la tasa |
| El capítulo argumenta que… | En este capítulo argumentamos que… |

**Regla de reparto de agencia:** las acciones cognitivas (analizar, interpretar,
diseñar, concluir, proponer) corresponden a los autores. Los modelos, ecuaciones
y variables solo ejecutan funciones algorítmicas: **estiman, calculan, predicen,
arrojan, producen**.

```
Mal:  "El modelo SAR decidió que la distancia era relevante."
Bien: "El modelo SAR estimó un coeficiente significativo para la distancia;
       concluimos que es una variable relevante."
```

Fórmulas de uso generalizado aceptadas: "los resultados sugieren", "los datos
proporcionan", "la Tabla 3 muestra".

### Matriz de asignación funcional de acciones verbales

| Sujeto | Verbos que le corresponden |
|--------|----------------------------|
| **Investigador** (voz activa) | analizamos, extrajimos, especificamos, modelamos, evaluamos, recopilamos, interpretamos, concluimos, proponemos |
| **Algoritmo / modelo** | calcula, estima, pondera, converge, minimiza la varianza, reporta, opera, predice |
| **Datos / tablas / mapas** | muestran, detallan, exhiben, exponen, constan, revelan |

```
Mal:  "El modelo SAR/SEM se encargó de analizar los errores espaciales."
Bien: "Utilizamos el modelo SAR/SEM para analizar los errores espaciales."
      (el modelo estima; nosotros analizamos)
```

## Adverbios introductores innecesarios

Eliminar o reestructurar:
- "Interesantemente, los resultados…" → "Los resultados revelan que…"
- "Notablemente, el CHC presenta…" → "El CHC presenta, de manera destacada,…"
- "Sorprendentemente, la variable…" → Reformular para que la oración
  lleve el peso del énfasis

## Contracciones y coloquialismos (prohibidos en texto académico)

| Evitar | Usar |
|--------|------|
| Bastante significativo | Estadísticamente significativo (p < .05) |
| Muy alto | Elevado / superior a [valor] |
| Casi todos | El 94% / la mayoría (n = 460) |
| Se ve que | Los datos indican que / se observa que |
| Hay que tener en cuenta | Es necesario considerar |

## Neutralidad descriptiva — sin adjetivación subjetiva

El discurso científico se sostiene sobre evidencia verificable, no sobre
valoración. Eliminar adjetivos y adverbios de impacto emocional, dramático,
hiperbólico o literario. La adjetivación debe ser **especificativa** (describe
un hecho observable o un parámetro), nunca **valorativa**.

| Subjetivo / hiperbólico | Neutro / verificable |
|-------------------------|----------------------|
| un resultado sorprendente | un resultado no anticipado por la hipótesis |
| una correlación impresionante | una correlación fuerte (r = .81) |
| un dramático aumento de precios | un aumento del 38% en los precios |
| la abrumadora evidencia | la evidencia (n = 487; 5 de 6 coeficientes significativos) |
| un hallazgo crucial / fascinante | un hallazgo relevante para [objetivo] |
| la innegable importancia del CHC | la concentración de valor en el CHC (β = .42) |
| claramente / obviamente / sin duda | [eliminar; que el dato sostenga la afirmación] |

**Contraste respetuoso pero no combativo:** al discutir a otros autores, usar
formulaciones neutras.

```
Combativo:  "Pérez (2019) pasó por alto completamente el efecto espacial."
Profesional: "Pérez (2019) no incorporó la dimensión espacial en su modelo."
```

## Verbos de atribución — precisión semántica

Cada verbo de atribución tiene matices distintos. Usar el correcto:

| Verbo | Cuándo usarlo |
|-------|--------------|
| propuso / planteó | teoría o marco conceptual |
| encontró / identificó | hallazgo empírico |
| demostró / comprobó | resultado con evidencia sólida |
| sugirió / indicó | resultado con cierto grado de incertidumbre |
| señaló / destacó | énfasis en un punto dentro de un argumento más amplio |
| argumentó / sostuvo | postura teórica o normativa |
| reportó | solo para resultados cuantitativos concretos |
| concluyó | afirmación final de un argumento completo |

**Evitar:** "menciona", "dice", "comenta" → impreciso y coloquial.

## Orden de modificadores en español

En español académico, el adjetivo evaluativo va DESPUÉS del sustantivo,
el especificativo también va después (salvo excepciones fijas):

```
Débil:  "la importante influencia espacial del patrimonio histórico"
Fuerte: "la influencia espacial del patrimonio histórico, que resulta
         significativa (β = .28, p < .001)"
```

Los adjetivos cuantitativos van ANTES: "los dos modelos", "las cinco variables".

---

## Conectores causales — precisión

En español, "ya que", "puesto que", "dado que" y "debido a que" expresan
causa. No son intercambiables con "como" en todos los contextos:

```
Ambiguo: "Como los precios varían espacialmente, se estimó SAR."
Preciso: "Dado que los precios exhiben autocorrelación espacial significativa
          (Moran's I = 0.34, p < .001), se estimó un modelo SAR."
```

## Conjunciones de tiempo y contraste — precisión semántica (APA §4.22)

En inglés APA prohíbe usar "while" para contraste (debe ser "although") y
"since" para causa (debe ser "because"). El principio aplica al español:

| Uso incorrecto | Corrección |
|----------------|-----------|
| "Mientras el CHC concentra valor, las periferias lo pierden" (contraste) | "Aunque el CHC concentra valor, las periferias lo pierden" |
| "Mientras se aplicó GWR, los coeficientes variaron localmente" (simultaneidad) | ✓ Correcto — "mientras" solo para eventos simultáneos |
| "Como los precios tienen sesgo, se aplicó log" (causa) | "Dado que los precios presentan sesgo positivo, se aplicó transformación logarítmica" |
| "Como mencionamos antes…" | "Tal como se indicó anteriormente…" |

**Regla:** "mientras" = simultaneidad; "aunque" = contraste; "dado que / puesto que / ya que" = causa (no "como").

## Modificadores mal colocados y colgantes (APA §4.23)

**Modificador colgante (dangling modifier):** el sujeto del modificador no
coincide con el sujeto principal de la oración.

```
Mal:  "Al aplicar el modelo SAR, los residuos mostraron independencia espacial."
      → Quien aplica el modelo son los investigadores, no los residuos.
Bien: "Al aplicar el modelo SAR, encontramos que los residuos mostraron
       independencia espacial."
Bien: "Los residuos presentaron independencia espacial una vez estimado el modelo SAR."
```

**Modificador mal colocado (misplaced modifier):** ambigüedad sobre qué
modifica el modificador.

```
Ambiguo: "Solo se analizaron los predios con uso residencial en el CHC."
         → ¿Solo se analizaron en el CHC, o solo los de uso residencial?
Preciso: "Se analizaron únicamente los predios de uso residencial ubicados
          en el CHC."
```

**Modificador oscilante (squinting):** puede modificar lo que va antes o lo
que sigue.

```
Ambiguo: "Los modelos que se usan frecuentemente producen errores estándar sesgados."
Preciso: "Los modelos de uso frecuente producen errores estándar sesgados."
```

**Etiqueta de error:** `[MODIF-COLGANTE]`, `[MODIF-AMBIGUO]`

## Modo verbal — indicativo vs. subjuntivo (APA §4.14)

- **Indicativo:** para hechos, resultados y afirmaciones empíricas.
- **Subjuntivo:** solo para condiciones contrarias a los hechos o hipotéticas.
- **Evitar "podría" o "debería" como evasión epistémica** sin justificación:

```
Evasivo:  "Los resultados podrían indicar que la accesibilidad influye en los precios."
Directo:  "Los resultados indican que la accesibilidad influye en los precios."
```

Si genuinamente hay incertidumbre, cuantificarla o citar respaldo:
```
Bien: "Estos resultados son consistentes con la hipótesis de capitalización
       del patrimonio (Ahlfeldt & Maennig, 2010), aunque un diseño
       cuasi-experimental sería necesario para establecer causalidad."
```

## Concordancia sujeto-verbo: sustantivos colectivos (APA §4.15)

En español, los sustantivos colectivos llevan verbo singular:
"la muestra incluye", "el grupo presenta", "el conjunto de datos muestra".

**Con sujetos compuestos unidos por "o" / "ni":**
el verbo concuerda con el sujeto más cercano:
```
"Ni el modelo SAR ni el modelo SEM captura toda la dependencia espacial."
→ Concuerda con "el modelo SEM" (el más próximo).
```

## Párrafos sin oración tópica — cómo detectarlo

Señal de alerta: el párrafo empieza directamente con "En el año X..." o
con un dato/cita sin declarar primero qué argumento sostiene esa evidencia.

Corrección: añadir una oración tópica de 15–25 palabras que declare la
afirmación central antes de presentar la evidencia.

```
Antes (sin oración tópica):
  "En 2023 se recolectaron 487 listings de portales inmobiliarios. La
   distribución de precios mostró asimetría positiva (sk = 1.43)..."

Después (con oración tópica):
  "Los datos de precios presentan la distribución asimétrica típica de
   los mercados inmobiliarios andinos. En 2023 se recolectaron 487 listings
   de portales inmobiliarios; la distribución exhibió asimetría positiva
   (sk = 1.43)..."
```
