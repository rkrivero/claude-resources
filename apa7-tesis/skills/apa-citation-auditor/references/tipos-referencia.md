# Tipos de referencia frecuentes en arquitectura e investigación urbana — APA 7, Cap. 10

## Artículo de revista (con DOI)

```
Autor, A. A., & Autor, B. B. (Año). Título del artículo en sentence case.
    Nombre de la Revista en Title Case, volumen(número), pp–pp.
    https://doi.org/xxxxx
```

Ejemplo:
```
Rosen, S. (1974). Hedonic prices and implicit markets: Product differentiation
    in pure competition. Journal of Political Economy, 82(1), 34–55.
    https://doi.org/10.1086/260169
```

## Artículo de revista (sin DOI, con URL)

```
Autor, A. A. (Año). Título. Nombre de la Revista, vol(núm), pp–pp.
    https://www.sitio.com/url
```

## Libro completo

```
Autor, A. A. (Año). Título del libro en cursiva. Editorial.
```

Nota: APA 7 elimina el lugar de publicación.

```
Anselin, L. (1988). Spatial econometrics: Methods and models. Kluwer Academic.
```

## Capítulo en libro editado

```
Autor, A. A., & Autor, B. B. (Año). Título del capítulo. En E. Editor
    (Ed.), Título del libro en cursiva (pp. xx–xx). Editorial.
    https://doi.org/xxxxx
```

## Informe técnico / gubernamental

```
Organización. (Año). Título del informe (Número de reporte si existe).
    Institución publicadora. https://url
```

```
Instituto Nacional de Estadística e Informática. (2017). Censos nacionales
    2017: XII de población, VII de vivienda y III de comunidades indígenas.
    INEI. https://censos2017.inei.gob.pe
```

## Tesis / disertación (repositorio institucional)

```
Autor, A. A. (Año). Título de la tesis [Tesis de licenciatura/maestría/doctorado,
    Universidad]. Repositorio institucional. https://url
```

## Página web con autor individual

```
Autor, A. A. (Año, Mes Día). Título de la página. Nombre del sitio.
    https://url
```

## Página web sin fecha

```
Organización. (s.f.). Título de la página. https://url
```

## Dataset / datos espaciales

```
Autor o Institución. (Año). Nombre del dataset [Conjunto de datos].
    Repositorio. https://doi.org/xxxxx
```

```
OpenStreetMap contributors. (2024). OpenStreetMap [Conjunto de datos].
    OpenStreetMap Foundation. https://www.openstreetmap.org
```

## Software

```
Autor, A. A. (Año). Nombre del software (Versión X.X) [Software].
    Editor / Empresa. https://url
```

```
Anselin, L. (2023). GeoDa (Versión 1.22) [Software]. GeoDa Center.
    https://geodacenter.github.io
```

Paquetes de Python / R (con versión):
```
Rey, S. J., Anselin, L., Amaral, P., Arribas-Bel, D., Cortes, R. X.,
    Gaboardi, J. D., Kang, W., Li, Z., Wolf, L. J., & Yunzhe, L. (2022).
    PySAL: A Python library of spatial analytical methods. Geographical
    Analysis, 54(3), 467–487. https://doi.org/10.1111/gean.12316
```

```
Bivand, R. S., & Wong, D. W. S. (2018). Comparing implementations of
    global and local indicators of spatial association. TEST, 27(3),
    716–748. https://doi.org/10.1007/s11749-018-0599-x
```

```

```
Anselin, L. (1995). SpaceStat: A software package for the analysis of
    spatial data (Versión 1.80). Regional Research Institute, West
    Virginia University.

## Libro editado (editor como autor principal)

```
Editor, E. E. (Ed.). (Año). Título del libro. Editorial.
    https://doi.org/xxxxx
```

```
Batty, M. (Ed.). (2013). The new science of cities. MIT Press.
```

## Traducción

```
Apellido, N. (Año). Título en español (T. Traductor, Trad.; N.ª ed.).
    Editorial. (Obra original publicada en año)
```

```
Lefebvre, H. (2013). La producción del espacio (E. Martínez Gutiérrez,
    Trad.). Capitán Swing. (Obra original publicada en 1974)
```

Cita en texto: `(Lefebvre, 1974/2013)`

---

## Entrada en enciclopedia / diccionario online (con autor)

```
Autor, A. A. (Año). Título de la entrada. En Título de la enciclopedia.
    Editorial. https://url
```

Sin autor → el título sube a posición de autor:
```
Renta del suelo. (2023). En Enciclopedia de Economía Urbana. Urban Economics Press.
    https://www.economiaurbanaweb.org/renta-del-suelo
```

## Ponencia de congreso publicada en actas

```
Autor, A. A., & Autor, B. B. (Año). Título de la ponencia. En E. Editor
    (Ed.), Título de las actas (pp. xx–xx). Editorial/Organización.
    https://doi.org/xxxxx
```

## Ponencia no publicada

```
Autor, A. A. (Año, Mes). Título [Tipo de presentación]. Nombre del Congreso,
    Ciudad, País.
```
